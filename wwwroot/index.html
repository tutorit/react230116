Welcome to the course

Seeing this page means that you have succeeded in setting up the course environment:

  • You have node-based web server running in the "server"-directory.
  • wwwroot-directory is the distribution directory of your web-server. This page is the index-html located in the wwwroot-directory.

JSX

The compiled React-applications should be placed to the "jsx"-directory. It already contains some demonstration applications.

To compile the jsx-files you should run:

>npm test
---Or you may install gulp globally----
>npm install -g gulp
---After which you may run---
>gulp jsx
